Secure Password Manager & Vault

Python Tkinter GUI Cryptography Local Storage

The Challenge

Managing unique, complex passwords for dozens of services is a security bottleneck for most users. I developed this desktop application to provide a secure, local-first solution for generating and storing credentials without relying on third-party cloud providers.

Application Workflow

The application follows a strict state-machine logic to ensure data remains locked until authorized. As shown in the diagram, users must pass a Login/Register gate before the Main Menu is accessible. Operations like Retrieval and Deletion are further protected by PIN verification.

üîí Cybersecurity Implementation

  • Local Encryption: Passwords are not stored in plain text. The application uses symmetric encryption to protect the local database.
  • Zero-Trust Retrieval: Even after logging in, a secondary PIN is required to copy or view a specific password, protecting against "session hijacking" if the computer is left unattended.
  • Input Filtering: Integrated "Exclude Characters" logic to ensure compatibility with various web-service password requirements.
