Homelab Infrastructure Project

This project is a self-hosted homelab running on Ubuntu with Docker-based services and a VPN-first architecture using Tailscale. The goal is to create a secure, private, and scalable home infrastructure environment.

Network Architecture

System Design

  • VPN-first access using Tailscale (no public exposure)
  • All services containerized using Docker
  • Pi-hole for network-wide DNS filtering
  • Nextcloud for private file storage
  • Home Assistant for IoT automation
  • Ollama for local AI inference

Traffic Flow

Personal Devices
   ↓
Tailscale VPN
   ↓
Ubuntu Server 
   ↓
Docker Containers
    ├── Nextcloud
    ├── Home Assistant
    ├── Ollama
    └── Pi-hole
