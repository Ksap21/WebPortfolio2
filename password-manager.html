Secure Password Manager

Python Tkinter GUI Local Encryption

Project Overview

This desktop application provides a secure, local vault for credential management. Instead of cloud storage, it emphasizes user-controlled data, utilizing a master authentication layer to protect a local JSON database.

System Logic Flow

The application is built as a state machine. Access to sensitive operations (Update/Delete) requires the re-entry of a unique PIN, ensuring that a simple left-open session does not compromise the entire vault.

Feature Gallery

Secure Entry: Multi-user registration and login gate.

Main Menu: Centralized hub for credential management.

Generator: Customizable entropy settings for strong passwords.

PIN Access: Secondary auth for retrieving stored keys.

Modification: Secure editing of existing entries.

Safe Removal: PIN-verified deletion to prevent accidents.
