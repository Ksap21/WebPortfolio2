Secure Password Manager

Python Tkinter GUI Local Encryption

Project Overview

This desktop application provides a secure, local vault for credential management. Instead of cloud storage, it emphasizes user-controlled data, utilizing a master authentication layer to protect a local JSON database.

System Logic Flow

The application is built as a state machine. Access to sensitive operations (Update/Delete) requires the re-entry of a unique PIN, ensuring that a simple left-open session does not compromise the entire vault.

Feature Gallery

Secure Entry Multi-user registration and login gate to prevent unauthorized local access.

Main Menu Centralized hub for credential management with a streamlined terminal aesthetic.

Generator Customizable entropy settings for strong passwords, allowing for specific character exclusions.

PIN Access Secondary authentication layer required specifically for retrieving stored sensitive keys.

Modification Secure editing of existing entries with validation checks before saving to the JSON database.

Safe Removal PIN-verified deletion process to prevent accidental loss of important credentials.
